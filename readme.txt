Tux Lectores Vip v1b.

Una extensión para Joomla 1.7/2.5 del tipo plugin de contenidos.
Sirve para permtir el acceso a los contenidos solo a usuarios definidos.

Está enfocada en novatos que aún no conocen o no saben utilizar correctamente el ACL nativo de la nueva verisón de Joomla.
