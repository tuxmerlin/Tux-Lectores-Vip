Tux Lectores Vip v1b.

Una extensiónn para Joomla 1.7/2.5 del tipo plugin de contenidos.
Sirve para permitir el acceso a los contenidos solo a usuarios definidos.

Está enfocada en novatos que aún no conocen o no saben utilizar correctamente el ACL nativo de la nueva versión de Joomla.
